介绍

Promise 对象是 JavaScript 中一个标准的内置对象, 用于表示一个异步操作最终完成状态结果.
Promise 的构造器接受一个 any 值, 这个值由执行器返回

Promise 存在下面三状态
待定 pending
成功 fulfilled
失败 rejected

Promise 对象是一个代理对象, 被代理的值在对象创建之初是未知的, 但是它允许你为异步操作的结果绑定不同的处理方法.

(resolve, reject) => {...} 带有两个参数, 它将在 Promise 构造执行结果返回前调用. 一般在执行器的内部进行一些异步操作, 并且通过 resolve 和 reject 两个参数将 Promise 的状态改成 fulfilled 或 rejected. 如果状态为 rejected 时, 执行函数的返回结果将会被忽略.

当状态被改变时, Promise 对象中的 then 中绑定的某个 handler 将会被调用, 同时, 因为 Promise.prototype.then 和 Promise.prototype.catch 都返回 promise 的对象, 他们可以被链式调用.

方法

Promise 的方法不多, 主要介绍这两个:

all

Promise.all(iterable)
返回新的 Promise, 这个对象会在 iterable 都成功才会成功, 并且返回同顺序的结果集. 如果错误的话, 只会返回第一个错误的结果.

race

Promise.race(iterable)
iterable 每次成功或者失败, 父 Promise 都会返回子 Promise 的成功返回值或失败详情作为参数调用父promise绑定的相应句柄, 并返回该promise对象.

使用
