Git进行GPG签名出现找不到密钥的问题
发表于 2020-12-21| 更新于 2021-01-13
阅读量:

问题

Windows 平台下, Kleopatra 生成的 GPG 在使用 Git 进行签名时出现无法签名的问题
附: 签名过程: 本地使用私钥签名 -> 托管平台使用公钥验证

错误信息

发生原因

当使用 Kleopatra 生成密钥时, 默认的存储目录为 AppData 内

解决方法

通过 config 指定 gpg 的工作目录即可解决该问题
